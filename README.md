# Barber Shop Website

Modern responsive website for a barber shop in Poland.

## Live Demo

https://royal-barbering.vercel.app

## About

This project is a landing page for a local barber shop.  
The website presents services, prices, gallery, contact information, opening hours, Google Maps link, Booksy booking and WhatsApp contact.

## Features

- Responsive mobile-first design
- Service and price list
- Booking buttons connected to Booksy
- WhatsApp contact button
- Google Maps link
- Photo gallery
- Opening hours section
- SEO metadata and social preview image

## Tech Stack

- Next.js
- TypeScript
- CSS
- Vercel
